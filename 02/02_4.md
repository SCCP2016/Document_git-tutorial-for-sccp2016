Chapter 2.4 - Installation - Other
=======

使用しているOSやLinuxディストリビューション、パッケージマネージャに応じてコマンドを入力しインストールしてください。

インストールコマンドの実行にはスーパーユーザー権限が必要になるので、必要に応じて `$ su` を実行するなりコマンドの先頭に `sudo` を付けるようにするなりしてください。

```
例)

/* Debian, Ubuntu, etc... */
$ apt-get install git

/* Fedora, RHEL, etc... */
$ yum install git
```
